# 静かなアラーム時計

会社で静かに使える、表示だけのシンプルな時刻通知アプリです。

## 特徴

- 音を鳴らさず、画面表示だけで通知
- 毎日アラームと単発アラームの両方に対応
- 「○分前に通知」機能で予定時刻から逆算して登録可能
- 設定は JSON ファイルに自動保存（終了しても消えない）
- PC ログイン時に最小化で自動起動

---

## 使い方

### 起動

`dist\静かなアラーム時計.exe` をダブルクリックして起動します。

### メイン画面

| エリア | 内容 |
|--------|------|
| 上部（濃紺） | 現在日時（1秒ごと更新） |
| 「毎日アラーム」タブ | 毎日同じ時刻に鳴るアラーム一覧 |
| 「単発アラーム」タブ | 特定日時に1回だけ鳴るアラーム一覧 |

### アラームの追加

1. 対象のタブ（毎日 / 単発）を開く
2. **追加** ボタンをクリック
3. 項目を入力して **登録** をクリック

#### 毎日アラーム

| 項目 | 内容 |
|------|------|
| 時・分 | 通知する時刻 |
| メッセージ | 通知に表示するテキスト |
| 有効 | チェックを外すと通知しない |

#### 単発アラーム（直接入力）

| 項目 | 内容 |
|------|------|
| 日付 | `YYYY/MM/DD` 形式で入力 |
| 時・分 | 通知する時刻 |
| メッセージ | 通知に表示するテキスト |

#### 単発アラーム（○分前に通知）

予定時刻から逆算して通知時刻を自動計算します。

入力例：
- 予定日: `2026/04/15`
- 予定時刻: `15:00`
- 何分前: `10`
- メモ: `来訪`

→ `14:50` に「`15:00 来訪の10分前です`」と通知されます。

### アラームの編集・削除

一覧でアラームを選択してから、右側のボタンを操作します。

| ボタン | 動作 |
|--------|------|
| 編集 | 時刻・メッセージを変更 |
| 削除 | アラームを削除（確認あり） |
| 有効化 | 無効にしたアラームを再び有効にする |
| 無効化 | 一時的に通知をオフにする |

### 通知ポップアップ

時刻になると画面中央に通知ウィンドウが最前面表示されます。

- 音は鳴りません
- **閉じる** ボタンを押すまで表示され続けます
- 単発アラームは通知後に自動で「完了」に変わります

## 自動起動の設定

PC ログイン時に最小化で自動起動するよう、Windows レジストリに登録済みです。

```
HKCU\Software\Microsoft\Windows\CurrentVersion\Run
値: "C:\PROJECT\clock\dist\静かなアラーム時計.exe" --minimized
```

### exe を移動した場合

レジストリの登録パスを更新する必要があります。
PowerShell で以下を実行してください（パスは実際の場所に合わせる）。

```powershell
Set-ItemProperty `
  -Path 'HKCU:\Software\Microsoft\Windows\CurrentVersion\Run' `
  -Name '静かなアラーム時計' `
  -Value '"C:\新しいフォルダ\静かなアラーム時計.exe" --minimized'
```

### 自動起動を解除したい場合

PowerShell で以下を実行します。

```powershell
Remove-ItemProperty `
  -Path 'HKCU:\Software\Microsoft\Windows\CurrentVersion\Run' `
  -Name '静かなアラーム時計'
```

## データ保存

アラーム設定は exe と同じフォルダの `alarms.json` に保存されます。

```json
{
  "daily_alarms": [
    {
      "hour": 11,
      "minute": 52,
      "message": "昼飯の時間です",
      "enabled": true,
      "last_triggered_date": "2026-04-15"
    }
  ],
  "one_time_alarms": [
    {
      "date": "2026-04-15",
      "hour": 14,
      "minute": 50,
      "message": "15:00 来訪の10分前です",
      "enabled": true,
      "done": false
    }
  ]
}
```

- `alarms.json` を手動で編集することも可能です
- exe を別のフォルダに移動する場合は `alarms.json` も一緒に移してください

---

## 動作仕様

| 項目 | 仕様 |
|------|------|
| 監視間隔 | 1秒ごと |
| 毎日アラーム | 同じ日に2回は通知しない |
| 単発アラーム | 通知後に自動で完了。再通知なし |
| 起動前に過ぎた通知 | 通知しない（起動時刻より前の時刻は無視） |
| 音 | なし |
| スヌーズ | なし |

## 開発・ビルド

### 必要なもの

- Python 3.12+
- PyInstaller 6.x

### exe のビルド

```bat
build.bat
```

または手動で：

```bat
pip install pyinstaller
pyinstaller --onefile --windowed --name "静かなアラーム時計" clock.py
```

ビルドされた exe は `dist\静かなアラーム時計.exe` に出力されます。
